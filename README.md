# Magno Música - Landing Page Premium

Uma landing page cinematográfica e de alta conversão desenvolvida para o Professor Magno Música, utilizando o que há de mais moderno no ecossistema Web (React 19, GSAP, TailwindCSS).

## ✨ Visão Geral

O projeto foi construído sob o conceito **"Midnight Luxe"**, combinando uma estética luxuosa de tons escuros com acentos dourados e interatividade fluida. O objetivo central é transformar visitantes em alunos através de uma narrativa visual envolvente e pontos de conversão estratégicos.

## 🚀 Tecnologias Utilizadas

- **Core:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Estilização:** [TailwindCSS](https://tailwindcss.com/)
- **Animações:** [GSAP](https://greensock.com/gsap/) (ScrollTrigger, MatchMedia)
- **Scroll Suave:** [Lenis](https://lenis.darkroom.engineering/)
- **Monitoramento:** Google Analytics 4 (GA4) com suporte à LGPD
- **Ícones:** Lucide React

## 💎 Diferenciais Técnicos

### 1. Performance & SEO (AEO Ready)
- Estrutura semântica completa (HTML5).
- Metadados otimizados para mecanismos de busca e assistentes de voz.
- Lazy loading de componentes e imagens.

### 2. Acessibilidade (A11y)
- Suporte total a navegação por teclado.
- Árias lables e contraste de cores validado.
- Foco gerenciado em modais e elementos interativos.

### 3. Analytics & LGPD
- Sistema de consentimento de cookies customizado.
- Rastreamento de eventos de conversão (Cliques em CTAs, Envio de Formulários, Abertura de Modais).
- Integração condicional do GA4 baseada no consentimento do usuário.

### 4. Cross-Browser & Responsividade
- Otimizado para Safari (iOS/macOS), Firefox e Chrome.
- Layouts adaptáveis de 320px a 2560px.
- Fix de Scroll Lock para dispositivos iOS.

## 🛠️ Instalação e Uso

1. **Clonar o repositório:**
   ```bash
   git clone https://github.com/DuardoHenrique/magno-musica.git
   ```

2. **Instalar dependências:**
   ```bash
   npm install
   ```

3. **Configurar variáveis de ambiente:**
   Crie um arquivo `.env` na raiz baseado no `.env.example`:
   ```bash
   VITE_GA4_ID=G-XXXXXXXXXX
   ```

4. **Rodar em desenvolvimento:**
   ```bash
   npm run dev
   ```
